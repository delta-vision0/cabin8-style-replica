# Cabin8 Salon Website

This is a Vite + React + TypeScript project using Tailwind CSS and shadcn-ui.

## Getting started

Prerequisites:
- Node.js 18+ and npm

Install dependencies:
```sh
npm install
```

Run the dev server:
```sh
npm run dev
```

Build for production:
```sh
npm run build
```

Preview the production build:
```sh
npm run preview
```

## Tech stack
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui
